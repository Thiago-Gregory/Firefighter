# Projeto Final Firefighter
Repositório do projeto final de Jogos Multiplayer

# Autores do projeto:
- Leonardo Korzenowski Saraiva, Gabriel de Oliveira Pedroso, Thiago Gregory Araujo

# Concept do jogo:
	Jogo de simulação de bombeiros onde um bombeiro deve resgatar pessoas presas em uma casa pegando fogo. Ele deve vasculhar os cômodos da casa em busca destas pessoas. Seu tempo é limitado pela barra de oxigênio abaixa constantemente, e, se chegar ao fim, é game over. Portas podem ser abertas para entrar em cômodos, mas algumas estão trancadas e devem ser derrubadas para ganhar acesso ao local. As pessoas resgatadas devem ser levadas de volta ao início onde há a safe zone. Em modo multiplayer, haverá dois bombeiros, um será responsável por apagar o fogo para liberar caminho para o outro bombeiro que será quem resgata as pessoas.
